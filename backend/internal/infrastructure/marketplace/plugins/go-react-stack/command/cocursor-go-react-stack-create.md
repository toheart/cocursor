# Create Go+React Full-Stack Project

Use the `go-react-stack` skill to create a new Go+React full-stack project.

## Usage

Invoke the skill:

```
openskills read go-react-stack
```

Then follow the interactive workflow in the skill to:
1. Gather project information (name, module path, description, etc.)
2. Create project directory structure
3. Generate configuration files
4. Generate OpenSpec specification files (if enabled)
5. Generate example code
6. Initialize Git repository (optional)

The skill will guide you through the entire process with detailed instructions.
